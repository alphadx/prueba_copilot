# Gestión y Resolución de STT

## Objetivos
- Definir claramente los objetivos del Sprint 3.
- Establecer un marco de trabajo claro para la gestión de STT.

## Roles Involucrados
- **Product Owner:** Responde a las dudas del equipo y prioriza el backlog.
- **Scrum Master:** Facilita la comunicación entre los miembros del equipo.
- **Desarrolladores:** Encargados de implementar las tareas del sprint.

## Flujos Implementados
1. **Identificación del STT:** Proceso para identificar el problema a resolver.
2. **Prioritización:** Evaluación de los STT en base a su impacto y urgencia.
3. **Resolución:** Implementación de las soluciones propuestas durante el sprint.
4. **Revisión:** Reunión para evaluar el progreso y ajustar el plan si es necesario.
